## Insets
An Insets object is a representation of the borders of a container. It specifies the space that a container must leave at each of its edges.

#### Format:
``insets="[<int:top&gt;] [<int:left&gt;] [<int:bottom&gt;] [<int:right&gt;]"``

#### Default value:
``0 0 0 0``

#### Exemples:
``insets="1 1 1 1"``<br>
``insets="0 1"``<br>
